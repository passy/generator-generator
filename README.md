> [Yeoman](http://yeoman.io) generator for yeoman-generator.


## Install

```
$ npm install --global generator-yeoman-generator
```


## Usage

Create a new directory, and go into:
```
mkdir my-new-generator && cd $_
```

With [yo](https://github.com/yeoman/yo):

```
$ yo generator
```
